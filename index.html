SinglePlayer:
After your webcam started showing video, place a Yellow object in front of it and use as a controller to the racket, you will see that the racket moves when you move the yellow object in front of cammera. Then, place the racket behind the ball, and move and click your mouse inside the field of the game. The game will start. Enjoy.
Multiplayer:
Same instructions as Singleplayer but this time, if you be a host, you need to click the Host button, and copy the code that will pop up and send it to the second player. The second player will have to place the code send by you in the box that will appear after they clicked in the Guest button. After they placed the code, they have to click in the Okay,I Pasted it button then send to you the generated code in the box below the one that they copy/pasted your code and send it to you. You will have to place the code they send in the box below the code that was generated when you clicked the Host button and click in Try to Connect button. The connection will occour and if it connect you will see the Connected status below the playfield
